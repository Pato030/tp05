1¿Qué atributo se utiliza para especificar la URL a la que se enviarán los datos del formulario? a) method b) action c) target d) url

El atributo acción. Este atributo define el lugar donde los datos se envian. Su valor debe ser una dirección URL válida. Si no se proporciona este atributo, los datos serán enviados a la dirección URL de la página que contiene el formulario

10¿Qué son los métodos / verbos de petición HTTP? Dar ejemplos.

Estos verbos indican qué acción queremos realizar sobre el servidor y son GET, POST, PUT, PATCH, DELETE, HEAD, CONNECT, OPTIONS y TRACE. Cada uno indica una acción diferente a la que el servidor debe responder.
